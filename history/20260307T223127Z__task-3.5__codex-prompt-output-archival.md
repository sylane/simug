# Task 3.5: Codex prompt/output archival

## Objective
- Persist full Codex prompts and raw outputs with stable correlation IDs for failure diagnosis.

## Scope
- In: archival artifacts for Codex input/output linked to run/tick/attempt context.
- Out: protocol changes and issue-triage behavior changes.

## Key Decisions
- Add `agent.RunError` carrying `RawOutput`, plus `agent.RawOutputFromError`, so failure paths (command/protocol/terminal errors) still preserve raw payload for archival.
- Archive every Codex attempt in `.simug/archive/agent/<run_id>/tick-<tick_seq>/attempt-<n>/` with:
  - `prompt.txt`,
  - `raw_output.txt`,
  - `metadata.json` (run/tick/attempt/branch/head/error context).
- Emit `agent_archive` / `agent_archive_error` events so archive artifacts are discoverable from `events.log`.
